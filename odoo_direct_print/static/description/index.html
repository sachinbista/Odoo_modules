Configuration


In Settings --> Technical --> Actions --> Reports. (You need to be in debug mode in order to see this menu).

Choose whatever report that you want to use the feature on.

In the report form view, in tab Print, field Default Behavior has 2 options to choose:

Sent to Client: the default options for all the report, when click on Print the report, the report will be downloaded.

Open Print Dialog: when click on Print the report, the system will open the browser print dialog for user to print the report.



Change Default Behavior from Send to Client to Open Print Dialog.


Now the related report(s) will be printdirectly without downloading the PDF file
