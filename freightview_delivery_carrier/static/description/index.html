Odoo FreightView Shipping Integration

Make shipping easy with Odoo Freightview Shipping Integration!

The module integrates the FreightView shipping service with Odoo so the orders can be delivered through FreightView delivery through the backend. Odoo FreightView Shipping Integration helps to ship products from one place to another efficiently. It offers the choice to quickly process orders by creating a Freightview shipment label in the Odoo backend. Admin can compare and choose shipping prices, track order delivery, packaging options, etc from Odoo.

Technical Requirements for FreightView API Integration

  • Provider: FreightView
  • Additional information: The shipment details is uploaded from odoo to the freightview portal. Rate and Other Shipment operations are managed from the FreightView portal.
  • Used for: Shipping, and Tracking
  • API: REST Based - v2.0 for Auth
  • Get API access: Register with https://www.freightview.dev/login
    • Please ensure that you have a valid FreightView API keys and secret to integrate FreightView.

One-Step Integration With Odoo

  • Simple configuration to allow fast integration of FreightView with Odoo.
  • You only need valid account details from FreightView shipping.

Real-Time Shipment Generation

  • The Shipping is generated when the shipping carrier is selected from the freightView end.
  • Generate FreightView shipping labels in real-time with a click of a button.

Select And Specify Packaging Options

  • After packages are created, the admin can choose the shipping carriers through FrieghtView with their cost.
  • The module also generates the FreightView shipment label for the order in the Odoo backend.

Easily works with Odoo Backend for smooth shipment

  • The Odoo user can add the FreightView Shipping to order from the backend.
  • Admin can easily compare and book the shipping carrier for shipment.

Choose Or Add the FreightView Shipping Service Type

  • Select the type of services availed from FreightView Shipping among the given options.
  • Alternatively, you can create a new service type as per your requirement.

Set Default values for faster operations

  • The Odoo user can set default values for fields such as product weight, Units of Measure (UoM) for both FreightView and Odoo, packaging, etc.
  • The packaging type available again depends upon the service opted from FreightView.
1. Follow the path Inventory>>Shipping Methods. Click on FreightView Delivery
2. Now, set the integration level for the module including price computation
Note: Make sure to only switch to the 'live environment' when you have set up the module and are ready to go live with it on the backend.
3. Add the FreightView delivery credentials, i.e., User API Key, Account API Key, ID, Secret, and Grant Type.
4. Create and view the names of the carriers together with their respective codes.
5. Create a sales order and confirm it without adding the shipping.
6. Navigate to delivery and On the picking select “FreightView Delivery Carrier” as the carrier under the page “Additional Info”. After selecting Freight Delivery Carrier, then create a package and there will be a button (FREIGHTVIEW BOOK URL) that will be visible upon picking. Make the appropriate package on the picking and click on this button.
7. When clicking on the Button FREIGHTVIEW BOOK URL, redirected to the FreightView page, and then select the shipping carrier with its cost.
8. After Selecting the carrier, it will schedule the shipment and can update the stage.
9. FreightView activity status changes by clicking on the new button “Fetch Shipping Details”.
10. To update the shipment status automatically, add FreightView Shipping Scheduler.
