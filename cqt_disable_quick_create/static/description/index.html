Disable Quick Create Edit

Key Features


This module disables the Quick Create option on relational fields all over the application.
It also allows to disable the Create and Edit option on a per model basis.
Go to Settings - Technical - Database Structure - Models
Select your model
Check the box Disable Create and Edit
Then Reload the page, after that user can not create quick that model many2one field
